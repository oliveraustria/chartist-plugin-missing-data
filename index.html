Chartist Plugin: Missing Data

This page hosts a plugin for the awesome chart library Chartist, which handles a data set with missing data.

Example with no displayed point of missing data set



            var data = {
                 labels: ['Week1', 'Week2', 'Week3', 'Week4', 'Week5', 'Week6'],
                    series: [
                          [1, 2, 3, null, 5, 6]
                     ]
            };
            var chart = new Chartist.Line('#chart1', data, {
                plugins: [
                    ctMissingData({
                      showMissingDataPoints: false
                    })
              ]
            });
